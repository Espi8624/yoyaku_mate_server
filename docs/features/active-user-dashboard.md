# 機能仕様書: アクティブユーザーダッシュボード (Active User Dashboard)

本文書は、`yoyaku_mate_admin` 本社管理者ウェブに実装されたリアルタイムアクティブユーザーモニタリングシステム（Active User Dashboard）の機能仕様を説明します。

> 最終更新: 2026-07-14  
> 関連文書: [リクエストダッシュボード仕様書](./request-counter.md), [ADR-004: アクティブユーザートラッキング意思決定文書](../decisions/ADR-004-active-user-tracking.md), [実装詳細書: アクティブユーザートラッキング](../implementation/active-user-tracking.md)

---

## 1. 概要 (Overview)

プラットフォームの安定性とユーザーの反応を即座に把握するため、リアルタイム同時接続者数およびアクティブユーザーのトレンド（DAU、MAU）をモニタリングするバックオフィスダッシュボード画面です。

---

## 2. 画面構成および主要指標 (UI Metrics)

管理者ダッシュボード上部のサマリーカード形式で表示され、5秒周期で自動更新（HTTP Polling）されます。

### 2.1 上部サマリーカード (Metrics Cards)

* **CURRENT ACTIVE USERS (リアルタイム同時接続者数)**
  * **説明**: 現在プラットフォームを実際に操作し、APIリクエストを継続的に発生させているユーザー数です。
  * **反映基準**: 直近5分以内に最低1回以上APIリクエストを送信したユニーク機器（IP）の数です。
* **DAILY ACTIVE USERS (DAU - 本日の接続者数)**
  * **説明**: 本日1日間にサービスを利用した累計ユニークユーザー数です。
  * **反映基準**: 当日（00:00〜現在）までに1回以上接続履歴が記録された重複のないIPの数です。
* **MONTHLY ACTIVE USERS (MAU - 過去30日間の接続者数)**
  * **説明**: 過去30日間にサービスを利用した累計ユニークユーザー数です。
  * **反映基準**: 直近30日間に1回以上接続履歴が記録された重複のないIPの数です。

---

## 3. 今後の高度化ロードマップ (Roadmap)

主要指標が安定化した後、ビジネス分析およびインフラ対応力を最大化するために段階的な高度化を行います。

### 3.1 第2段階: ビジネス高度化指標
* **DAU/MAU比率 (ユーザーの粘着度 - Stickiness)**
  * 月間ユーザーのうち、毎日訪問するユーザーの比率を計算し、ユーザーのリテンションおよびロイヤルティの変化を分析します。
* **前日比増減率 (Growth Indicator)**
  * 本日の接続者数の下部に、前日の同時間帯に対するトラフィック増減推移をパーセント（%）で表示し、成長を直感的にトラッキングします。

### 3.2 第3段階: インフラおよびプラットフォーム分析
* **本日のピークタイムおよび最大同時接続者数 (Peak Concurrent Users & Time)**
  * 本日最もトラフィックが集中したピーク時間と最大同時接続者数を表示することで、サーバーのスケーリング計画の策定を支援します。
* **機器およびプラットフォーム別の接続比率 (Web vs App)**
  * アクティブユーザーのプラットフォーム分布（Web / Android App / iOS App）をドーナツチャート形式で可視化し、フロントエンド最適化の優先順位付けに活用します。
